# Description :books:
The tests are conducted on a e-commerce test store which is alike real stores.
Store URL: https://teststore.automationtesting.co.uk

# Technology Stack :bulb::
- Programming language: Java
- Automation framework: Selenide
- Testing framework: TestNG
- Project management tool: Maven
- Third-party libraries: Faker, ExtentReports

# Examples :scroll:
 - To view the test results, navigate to the "report" folder and open the test result report file in a web browser. This report is automatically generated after running the tests. It contains all the necessary logs and details regarding the test execution..
